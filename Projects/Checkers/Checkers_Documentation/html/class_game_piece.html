Detailed Description

The GamePiece class is an abstract base class representing a general game piece to be used in a game of checkers. The class contains five virtual functions for selecting and validating choices of: game squares for initial selecting of a piece, and moving the piece to the new square. Three classes are derived from GamePiece; RegBlack,RegWhite,and King. Each of these three classes require their own unique pointer arithmetic in order to give them the proper functionality with respect to their place in the game. Two non virtual methods are included also for checking for a double jump and swapping characters.
